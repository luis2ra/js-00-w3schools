10.- Ending Definitions with a Comma

Trailing commas in object and array definition are legal in ECMAScript 5.

Object Example: person = {firstName:"John", lastName:"Doe", age:46,}

Array Example: points = [40, 100, 1, 5, 25, 10,];

WARNING: Internet Explorer 8 will crash. JSON does not allow trailing commas.

JSON Object Example: person = {firstName:"John", lastName:"Doe", age:46}

JSON Array Example: points = [40, 100, 1, 5, 25, 10];
