JavaScript Date Methods

Date methods let you get and set date values (years, months, days, hours, minutes, seconds, milliseconds)

Date Set Methods

Set methods are used for setting a part of a date. Here are the most common (alphabetically): setDate(), setFullYear(), setHours(), setMilliseconds(), setMinutes(), setMonth(), setSeconds() and setTime().

The setFullYear() Method

The setFullYear() method sets a date object to a special date.

Remember that JavaScript counts months from 0 to 11. Month 10 is November.

The setDate() Method

The setDate() method sets the date of a month.

The setDate() method can be used to add days to a date.
