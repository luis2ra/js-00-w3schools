JavaScript Numbers

JavaScript will try to convert strings to numbers in numeric operations:

If x = "100" and y = "10".

This will work:
Case 1: z = x / y;

This will also work:
Case 2: z = x * y;

And this will work:
Case 3: z = x - y;

But this will not work:
Case 4: z = x + y;

In the last example JavaScript uses the + operator to concatenate the strings.
